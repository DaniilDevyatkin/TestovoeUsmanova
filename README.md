# TestovoeUsmanova
Реплика лендинга фитнес-тренера Кати Усмановой на jQuery + Bootstrap + GetCourse. Три экрана: Hero, «О тренере» и Social Proof (Desktop + Mobile). Pixel-perfect 1:1 с оригиналом. Локальные изображения, адаптивная вёрстка, кастомные стили кнопок, блоки с CMS-разметкой. Подходит как шаблон для GetCourse‑лендингов.
